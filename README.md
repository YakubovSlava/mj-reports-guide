# Инструкция по созданию отчётов

Это руководство для тех, кто никогда не писал код. Следуйте шагам по порядку — ничего лишнего знать не нужно.

---

## Шаг 1. Включите VPN

**Это нужно сделать до всего остального.**

Claude (искусственный интеллект, который будет помогать вам) работает через серверы в США. В некоторых сетях доступ к ним заблокирован — VPN решает эту проблему.

Включите VPN **до запуска** приложения Claude Code. Если VPN включить уже после — перезапустите приложение.

---

## Шаг 2. Скачайте папку с проектом

Вам не нужен Git или технические знания. Просто скачайте ZIP-архив:

1. Откройте ссылку на репозиторий в браузере
2. Нажмите зелёную кнопку **`<> Code`** (в правой верхней части страницы)
3. В выпадающем меню выберите **`Download ZIP`**
4. Сохраните архив на компьютер и **распакуйте** его в любую папку (например, на Рабочий стол)

После распаковки у вас будет папка `reports-guide` (или аналогичная). Именно её нужно открыть в следующем шаге.

---

## Шаг 3. Установите и подключите Claude Code

### Установка приложения

Скачайте **Claude Code** — это официальное приложение от Anthropic:

- Перейдите на сайт **claude.ai/download** и скачайте версию для вашей системы (macOS или Windows)
- Установите приложение как обычную программу

> **IDE (VS Code, PyCharm и т.п.) не нужны.** Claude Code — самостоятельное приложение, которое умеет запускать код само.

### Подключение аккаунта

При первом запуске Claude Code попросит войти в аккаунт:

- Если у вас есть подписка **Claude Pro** (claude.ai) — войдите через браузер, когда приложение откроет страницу авторизации
- Если у вас есть **API-ключ Anthropic** — вставьте его в поле `API Key` при первом запуске

> Если ни того ни другого нет — зарегистрируйтесь на **claude.ai** и оформите подписку Pro, либо получите API-ключ на **console.anthropic.com**.

---

## Шаг 4. Откройте проект и начните работу

1. Запустите **Claude Code**
2. Выберите пункт **Open Folder** (или перетащите папку `reports-guide` в окно приложения)
3. Укажите папку, которую вы распаковали на Шаге 2
4. Claude автоматически прочитает инструкции из папки и будет готов помочь
5. Напишите ему по-русски, какой отчёт вы хотите создать — например:

   > «Хочу таблицу с топом жалоб по продуктам за последний месяц»

   или

   > «Сделай воронку продаж по каналам с конверсиями»

Claude сам разберётся с остальным: напишет код, проверит его, покажет предпросмотр и подскажет, как загрузить результат в платформу.

---

## Как выглядит процесс создания отчёта

После того как вы описали задачу, Claude проведёт вас через несколько этапов. Вот что будет происходить:

### Этап 1 — Уточнение задачи

Claude может задать один уточняющий вопрос: например, за какой период нужен отчёт или нужны ли фильтры по продукту. Ответьте в свободной форме — технических знаний не требуется.

### Этап 2 — Тестовые данные

Прежде чем писать отчёт, Claude сгенерирует небольшой набор **тестовых данных** (синтетику). Это обязательный шаг: он проверяет, что отчёт правильно читает структуру файла.

**Важно:** после того как синтетика сгенерирована, проверьте её и убедитесь, что она **точно совпадает по структуре с вашими реальными данными** — те же колонки, те же типы значений, похожий формат дат. Если что-то расходится — скажите Claude, он исправит. Без этого отчёт может работать на тестах, но сломаться на боевых данных.

### Этап 3 — Написание скрипта

Claude напишет программный код отчёта и сохранит его в папку. Вам ничего делать не нужно — просто наблюдайте.

### Этап 4 — Предпросмотр

Claude автоматически запустит отчёт на тестовых данных и сформирует HTML-файл с предпросмотром. Он будет выглядеть так же, как в платформе.

> Откройте файл `my_report.html` в браузере (двойной клик) и посмотрите результат. Если что-то не так — опишите Claude, что именно, и он исправит.

### Этап 5 — Загрузка в платформу

Когда предпросмотр вас устраивает, Claude даст чёткую инструкцию: куда зайти в платформе MJ, какие поля заполнить и что вставить. Следуйте ей шаг за шагом.

## Что делать, если что-то не работает

| Проблема | Решение |
|----------|---------|
| Claude не отвечает или выдаёт ошибку соединения | Проверьте, включён ли VPN |
| Приложение просит войти снова | Войдите в аккаунт claude.ai повторно |
| Claude пишет об ошибке в коде | Просто покажите ему текст ошибки — он исправит сам |
| Не видно предпросмотра отчёта | Напишите Claude: «покажи превью» — он запустит сам |
